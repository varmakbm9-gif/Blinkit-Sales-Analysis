# 🛒 Blinkit Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents a **comprehensive sales analysis of Blinkit**, India's last-minute grocery delivery platform.
The dashboard was developed using **Power BI** to uncover meaningful insights about sales performance, item categories, outlet types, and customer ratings.

The objective of this project is to transform raw sales data into **interactive visual insights** that help understand business performance and decision-making patterns.

---

## 🎯 Objectives

* Analyze overall sales performance
* Understand the distribution of items across categories
* Compare outlet size and location performance
* Evaluate customer ratings and product visibility
* Identify trends that impact total sales

---

## 🛠 Tools & Technologies Used

* **Power BI** – Dashboard creation & visualization
* **Power Query** – Data cleaning and transformation
* **Excel / CSV Dataset** – Data source

---

## 📂 Dataset Description

The dataset contains sales information for Blinkit grocery products.
Key fields in the dataset include:

* Item Identifier
* Item Type
* Item Fat Content
* Outlet Size
* Outlet Location
* Outlet Type
* Item Visibility
* Sales
* Customer Ratings

---

## 🧹 Data Cleaning Steps

Data preprocessing was performed in **Power Query**:

* Removed duplicate records
* Handled missing values
* Standardized categorical values
* Corrected data types
* Filtered inconsistent entries

---

| Metric         | Value  |

---

## 📈 Dashboard Insights

* **Tier 3 outlets** contribute the highest sales compared to Tier 1 and Tier 2.
* **Fruits, snacks, and household products** are among the top selling categories.
* **Medium-sized outlets** generate the majority of the revenue.
* Most items have a **customer rating close to 4**, indicating good customer satisfaction.
